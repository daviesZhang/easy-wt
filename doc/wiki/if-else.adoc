.用例的步骤支持 分支判断
#当步骤类型选择了 `IF` 后续步骤一定要有 `END IF` 来结束分支#

当步骤类型选择 `IF` 时

* `元素选择器` 输入项不为空时

1. 元素选择器所选择的元素如果存在,则会执行 `IF` 至 `ELSE` 或 `END IF` 中间所包含的步骤

2. 元素选择器所选择的元素如果不存在,则不会执行中间步骤,但是如果 `IF` 步骤后面存在 `ELSE` 步骤,则会执行 `ELSE` 至 `END IF` 中间所包含的步骤

* `内容` 输入项不为空时

1. 内容输入项所描述的表达式成立,则会执行 `IF` 至 `ELSE` 或 `END IF` 中间所包含的步骤

2. 内容输入项所描述的表达式不成立,则不会执行中间步骤,但是如果 `IF` 步骤后面存在 `ELSE` 步骤,则会执行 `ELSE` 至 `END IF` 中间所包含的步骤

* 如果 `元素选择器` 和 `内容` 同时不为空,则两个条件都需满足
