.用例的步骤支持循环执行
#当步骤类型选择了 `WHILE` 后续步骤一定要有 `END WHILE` 来结束循环#

当步骤类型选择 `WHILE` 时

* `元素选择器` 输入项不为空时

元素选择器所选择的元素如果存在,则会重复执行 `WHILE` 至 `END WHILE` 中间所包含的步骤

* `内容` 输入项不为空时

内容输入项所描述的表达式成立,则会重复执行 `WHILE` 至 `END WHILE` 中间所包含的步骤

* 如果 `元素选择器` 和 `内容` 同时不为空,则两个条件都需满足,才会进入循环执行
